# 로또 앱

## 1 기능

#### 1 랜덤 로또 생성

* 이 앱은 랜덤한 숫자를 생성하는 작은 역할을 합니다.  
단순히 버튼을 누르는 것으로 작업을 수행할 수 있습니다.

#### 2 나눔로또 웹 페이지 제공 

* 나눔로또 웹 페이지를 제공합니다

#### 3 각 회별 당첨 정보를 제공 (미 구현)

* 나눔로또 웹 페이지에서 확인 가능하지만,  
크롤링, client database, visualization 연습용으로 작성할 것입니다.  
만약 데이터를 확인하고, 출현 횟수가 적은 숫자에 마음이 끌린다면 이 기능도 도움이 될거 같네요. : )  
금방 된다는 보장은 없습니다. 하지만 당분간은 여기에 힘을 쓸 것입니다.
